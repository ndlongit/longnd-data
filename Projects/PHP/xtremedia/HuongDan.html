XtreMedia 1.2

###### INSTALL ######
Làm theo các bước sau.

   1. Mở file ./includes/config.php và chỉnh sửa các thông số cho phù hợp
   2. CHMOD tất cả các file *.xml trong thư mục /shoutbox/data/ thành 777
   3. Chạy http://www.your_media_site.com/install.php để cài đặt Database cho XtreMedia
   4. Xóa file install.php ra khỏi Website của bạn.
